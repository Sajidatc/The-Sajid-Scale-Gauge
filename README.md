# The-Sajid-Scale-Gauge

The manuscript entitled "The Sajid Scale Gauge: Discovery 
of a New Fundamental Constant in Prime Number Theory".

This paper introduces a novel metric for analyzing prime gaps—the Sajid 
Gauge—defined as B = floor(g / floor(log2 p)). Through extensive 
computational verification up to 5 × 10^9, we:

1. Discover five new maximal prime gaps (B = 7 through B = 11)
2. Demonstrate a linear relationship B = κ · log10(p) + ε with R² = 0.9967
3. Identify a new fundamental constant κ_s = 1.115 ± 0.020
4. Independently verify OEIS sequence A005250

The computational work was performed on an Intel i7-8850H system with 
32GB RAM from Gilgit-Baltistan, Pakistan.
